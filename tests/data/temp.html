Příklad: Pro populární český cloud TMEP.CZ, cloud url může vypadat takto: http://brrrcz.tmep.cz/?temp= , kde brrrcz je parametry doména a temp je parametr GUID. Tyto parametry zadáte při vytváření čidla na serveru tmep.cz
Cloud nastavení: Cloud server zatím nenastaven

Možnost vypnutí zasílání dat na brrr.cz server

Pokud nechcete zasílat data na server brrr.cz, vyplňte formulář velkým tiskacím NE a odešlete. Jakékoliv jiné zadání, odesílání na brrr.cz server zase obnoví. Stav: Zasílání na brrr.cz vypnuto

Možnost vypnutí Wifi signálu teploměru po 5 minutách

Pokud chcete vypnout wifi signál teploměru po 5 minutách, vyplňte formulář velkým tiskacím ANO a odešlete. Stav: Wifi signál teploměru je po pěti minutách vypnut.

Nastavení hesla

Od výroby nastavuji wifi teploměr bez hesla. Zde můžete nastavit heslo. V případě ztráty hesla, nastavte HotSpot na mobilu: teplomerreset a teploměr znovu zasuňte a heslo se vymaže. Stav: Bez hesla
